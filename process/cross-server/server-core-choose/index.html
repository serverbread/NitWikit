关于 waterfall​

关于 Waterfall 停止维护的公告。

信息

停止维护是指不再为其添加新功能或错误修补，

Waterfall 仍然会合并来自 BungeeCord 的更改。

对于仍然想使用 Waterfall 的人来说这可能不是一个很大的影响。

目前为止，你可以选择以下核心进行转发：

名称介绍推荐与否下载镜像
WaterfallPapermc 对 BungeeCord 的分支，但已停止向其添加新功能不推荐McRes FastMirror 原子云
LightfallArclight waterfall的分支，支持forge玩家进入服务器-McRes FastMirror
Velocity新生代代理端核心，拥有高安全性和高性能，但是插件不如BungeeCord那边多非常推荐McRes(自己点进去找) FastMirror 原子云
BungeeCord最古老的代理端核心，甚至有网页版mc的核心-McRes FastMirror
Gate基于Go开发的核心，拥有超高的性能和极低内存占用。-

富哥的选择

名称介绍价格
NullCordX基于反机器人原生构建的复杂瀑布叉，提高了性能10美元
XCordBC叉：高级反机器人性能 反漏洞利用 Anti-SpigotExploit（1.7-1.20.4）
FlameCord终极反机器人解决方案，Anti-VPN和高性能BungeeCord分叉，适用于Minecraft服务器6美元（永久15美元）
